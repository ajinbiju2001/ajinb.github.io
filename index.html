I am a passionate Software Test Engineer with experience in both manual and automation testing. I specialize in ensuring web and mobile applications deliver high quality, reliability, and a seamless user experience. My testing approach focuses on identifying defects early, improving usability, and maintaining product stability through structured test scenarios and detailed validation.

I have hands-on experience with tools and technologies including Selenium WebDriver, Java, TestNG, JUnit, SQL, Postman, JMeter, Jira, and Git. I have worked on testing websites, mobile applications (iOS & Android), and e-commerce platforms — performing functional testing, UI testing, API testing, and automation scripting.
